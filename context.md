# Контекст проекта: Личная CRM фотографа

## Основная информация

- **Цель**: Простая система для учёта моих клиентов
- **Для кого**: Только для меня, один пользователь
- **Технологии**: React + Firebase
- **Статус**: Только начинаю, MVP этап
- **Локация**: Разработка из Беларуси

## Технический стек (предварительно)

- **Фронтенд**: React, возможно TypeScript позже
- **Стили**: Tailwind CSS или простой CSS
- **База данных**: Firebase Firestore
- **Аутентификация**: Пока не нужна (только я)
- **Хостинг**: Firebase Hosting (бесплатный)

## Что должно делать (общие требования)

1. ✅ Добавлять новых клиентов
2. ✅ Просматривать список всех клиентов
3. ✅ Искать клиентов по имени, email, номеру телефона и instagram аккаунт
4. ✅ Редактировать информацию о клиентах
5. ✅ Удалять клиентов

## Структура данных (БУДУ ДУМАТЬ ПОЗЖЕ)

Пока знаю только базовые поля:

- Имя клиента
- Телефон (необязательно)
- Email (необязательно)

Остальные поля добавлю в процессе разработки.
